🚗 First-Time Setup

Create your admin account to get started

Important: Before running this, temporarily set your Firestore rules to allow all writes. After creating the admin, update the rules to the secure version from firebase-rules.js.

After creating the admin:

  1. Go to Firebase Console > Firestore > Rules
  2. Replace rules with the secure version from firebase-rules.js
  3. Open index.html and log in with your new admin credentials
  4. You can now create more users from the Admin panel
